HTMLs Historia

HTML (Hypertext Markup Language) är det primära formatet som används för att skapa webbsidor. Det lanserades för första gången 1990 av Tim Berners-Lee, en brittisk ingenjör, med syftet att göra det möjligt för forskare att dela information med varandra. HTML-koden är en serie av taggar som beskriver innehållets layout och formatering. Genom åren har HTML utvecklats och idag är det den vanligaste standarden för att skapa och visa webbsidor. HTML har genomgått flera versioner sedan det först lanserades. Den första versionen, HTML 1.0, var mycket begränsad i sin funktionalitet och användes endast för att strukturera text. Efterföljande versioner har inkluderat fler taggar och funktioner för att möjliggöra mer avancerad webbutveckling. Idag är den senaste versionen HTML5, som innehåller många nya funktioner och förbättringar som gör webbutveckling ännu mer kraftfull och flexibel.

Hur fungerar Dokumentobjektmodellen (DOM)?

Dokumentobjektmodellen, eller DOM

Dokumentobjektmodellen (DOM) är en programmeringsgränssnitt som representerar HTML-dokumentet som en hierarkisk struktur av element och attribut. Med hjälp av DOM kan man ändra och manipulera webbsidans innehåll och utseende genom att ändra elementens egenskaper, lägga till nya element eller ta bort befintliga. DOM kan också användas för att dynamiskt uppdatera webbsidans innehåll utan att behöva ladda om hela sidan.

När en webbläsare laddar en HTML-sida, analyseras sidans kod och konverteras till en DOM-struktur som webbläsaren kan använda för att rendera sidan. Genom att manipulera DOM kan man skapa dynamiska och interaktiva webbsidor, till exempel genom att lägga till händelsehanterare som reagerar på användarens klick och muspekare.

I slutändan är det genom DOM som HTML och JavaScript samverkar för att skapa moderna webbsidor med avancerade funktioner och interaktioner.
